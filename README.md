# 人生コンパスタイプ診断 (Life Compass)

12問の質問に答えて、あなたの行動スタイルを8タイプに分類する診断サイトです。

## 起動手順

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

## 機能

- **トップページ** (`/`) - 診断の概要と開始ボタン
- **診断ページ** (`/quiz`) - 12問を1問ずつ表示、5段階回答
- **結果ページ** (`/result`) - タイプ名・強み・落とし穴・今日の一手を表示
- **診断についてページ** (`/about`) - 3軸の説明とスコア算出方法

## 技術スタック

- Vite + React (JavaScript)
- react-router-dom（ルーティング）
- プレーンCSS（追加CSSライブラリなし）

## ビルド

```bash
npm run build
```

`dist/` ディレクトリに静的ファイルが生成されます。
